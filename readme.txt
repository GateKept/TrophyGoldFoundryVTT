This is a basic implementation of Trophy Gold for Foundry VTT.

Special chat commands:

To perform a risk roll, type "/risk [Light Dice][Dark Dice]"
To roll for your Weakpoint, type "/weakpoint"
To roll for a round of combat, type "/combat [Dark Dice]"
To find out how much random loot you can get from a dead enemy, type "/loot [Light Dice]"
